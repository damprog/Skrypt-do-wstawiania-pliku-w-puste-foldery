<@> Program do wstawiania pliku do pustych folderów <@>

Najpierw należy skonfigurować plik ps.ps1
Aby uruchomić program należy kliknąc na plik z rozszerzeniem *.bat


		@KONFIGURACJA@
Przed uruchomieniem należy skonfigurować ps.ps1 > Edytuj - powinien uruchomić się edytor powershell
Potem należy zmienić ścieżkę do docelowego folderu nadrzędnego, w którym program będzie działał
Zmiany należy dokonać TYLKO w jednym miejscu:
$Path = "Tu podać odpowiednią ścieżkę"
